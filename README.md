# AI Course Lesson 1

第一堂課 HTML 簡報第一版。

## 使用方式

直接開啟 `index.html` 即可瀏覽。也可以放到 GitHub Pages。

## 操作

- 滾輪 / 觸控板：上下切換頁面
- 鍵盤：↑ ↓ PageUp PageDown Space
- 右下角按鈕：上一頁 / 下一頁
